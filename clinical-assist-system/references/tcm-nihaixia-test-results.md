## 测试1：感冒发烧（六经辨证）

**检索路径**: SKILL.md grep "感冒发烧" → L172-173 速查 + L395-749 太阳病篇条文详解

**结果**:
- 速查: ①有汗+怕风→桂枝汤 ②无汗+怕冷→麻黄汤 ③忽冷忽热→小柴胡汤 ④但热不寒→白虎汤
- 深层: 桂枝汤组方原理（桂枝壮心阳/白芍静脉回流/生姜刺激肠胃/大枣补津液/炙甘草解百毒）
- 剂量: 胖子五钱瘦子三钱小孩半钱；阳不足加重桂枝，阴不足加重白芍
- 煎服法: 大火煮取气，啜热稀粥助药力
- 伤寒论原文: 第1条、第12条

**评价**: ⭐⭐⭐⭐⭐ 速查表+条文详解+组方原理+剂量，层次完整

## 测试3：小柴胡汤方剂咨询

**检索路径**: SKILL.md grep "小柴胡汤" → L347 速查 + L3911-3980 条文105详解

**结果**:
- 组成: 柴胡、黄芩、人参、半夏、生姜、大枣、炙甘草
- 四大主证: 往来寒热、胸胁苦满、默默不欲食、心烦喜呕
- 伤寒论第105条完整原文
- 七味药药理: 柴胡半表半里入淋巴 / 黄芩去热 / 半夏治胸胁苦满 / 人参治心下满
- 临床场景: 小孩感冒→西医退烧药+镇静剂→淋巴结肿大→往来寒热 = 小柴胡汤证
- 柴胡药性: 一半在表一半在里→甲状腺/内分泌/淋巴结问题都用柴胡

**评价**: ⭐⭐⭐⭐⭐ 组成→条文→药理→临床场景，实用性强

## 测试4：糖尿病医案检索

**检索路径**: SKILL.md grep "糖尿病" → L6087 + L6667-6685 医案; modules/03_yian.md grep "糖尿病" → L122-140

**结果**:
- 分三消辨治: 上消→白虎汤 / 下消→肾气丸
- 医案1（050405）: 二型糖尿病=富贵病（宵夜+不运动），治法先强肾再护心
- 医案2（050419）: 56岁白人糖尿病12年，炮附子五钱+石膏五两（白虎+玄武同用），两周双脚恢复知觉
- 预后判断: 睡眠+胃口+二便+手足温度

**评价**: ⭐⭐⭐⭐ 理论+多个真实医案+具体剂量，但倪师对西医批评较激进

### 融合建议
1. 适合作为中医辅助诊断知识源灌入 ChromaDB `tcm_knowledge` 集合
2. 融合时必须加 disclaimer: "中医分析仅供参考，需执业中医师确认"

4. max_tokens 需从 2500 提至 3500-4000 以容纳 tcm_analysis 字段
